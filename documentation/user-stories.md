# 1️⃣ Guest (Unauthenticated User)

### Account

* As a **guest**, I want to **register an account** so that I can apply for jobs.
* As a **guest**, I want to **login** so that I can access my account.
* As a **guest**, I want to **reset my password** so that I can recover access if I forget it.
* As a **guest**, I want to **verify my email** so that my account becomes active.

### Job Browsing

* As a **guest**, I want to **browse job listings** so that I can see available opportunities.
* As a **guest**, I want to **search jobs by keyword** so that I can find relevant jobs.
* As a **guest**, I want to **filter jobs by location** so that I can find jobs near me.
* As a **guest**, I want to **filter jobs by job type** so that I can see remote or full-time jobs.
* As a **guest**, I want to **view job details** so that I can understand the requirements.

### Companies

* As a **guest**, I want to **view company profiles** so that I can learn about employers.
* As a **guest**, I want to **see jobs posted by a company** so that I can explore opportunities there.

---

# 2️⃣ Job Seeker

### Profile

* As a **job seeker**, I want to **create and update my profile** so employers can learn about me.
* As a **job seeker**, I want to **upload my CV** so that employers can review my experience.
* As a **job seeker**, I want to **add my education history** so that employers see my qualifications.
* As a **job seeker**, I want to **add my work experience** so that employers understand my background.
* As a **job seeker**, I want to **add skills** so that job matching becomes easier.

### Job Discovery

* As a **job seeker**, I want to **search jobs by title** so that I can find relevant jobs.
* As a **job seeker**, I want to **filter jobs by salary range** so that I can find jobs within my expectations.
* As a **job seeker**, I want to **filter jobs by experience level** so that I see jobs suitable for me.
* As a **job seeker**, I want to **sort jobs by date posted** so that I see the newest jobs first.

### Applications

* As a **job seeker**, I want to **apply for a job** so that employers can consider me.
* As a **job seeker**, I want to **attach a CV to my application** so employers can review my profile.
* As a **job seeker**, I want to **write a cover letter** so that I can introduce myself to employers.
* As a **job seeker**, I want to **track the status of my applications** so I know whether they are accepted or rejected.

### Job Management

* As a **job seeker**, I want to **save jobs** so that I can apply later.
* As a **job seeker**, I want to **remove saved jobs** so that I keep my list organized.
* As a **job seeker**, I want to **view jobs I have applied to** so I can track them.

### Notifications

* As a **job seeker**, I want to **receive notifications when my application status changes** so that I stay informed.
* As a **job seeker**, I want to **receive notifications for new jobs matching my skills**.

---

# 3️⃣ Employer / Company

### Company Profile

* As an **employer**, I want to **create a company profile** so that job seekers can learn about my company.
* As an **employer**, I want to **update company information** so that it stays accurate.
* As an **employer**, I want to **upload a company logo** so that my brand is visible.

### Job Posting

* As an **employer**, I want to **create a job post** so that job seekers can apply.
* As an **employer**, I want to **edit a job post** so that I can update job details.
* As an **employer**, I want to **delete a job post** so that outdated jobs are removed.
* As an **employer**, I want to **close a job post** so that no new applications are accepted.

### Applicant Management

* As an **employer**, I want to **view applicants for a job** so that I can review candidates.
* As an **employer**, I want to **view applicant CVs** so that I can evaluate their qualifications.
* As an **employer**, I want to **accept or reject applicants** so that I can manage hiring.
* As an **employer**, I want to **shortlist candidates** so that I can review them later.

### Job Analytics

* As an **employer**, I want to **see the number of applications** so that I know how popular the job is.
* As an **employer**, I want to **see the number of job views** so that I know how visible the job is.

---

# 4️⃣ Admin

### User Management

* As an **admin**, I want to **view all users** so that I can monitor the platform.
* As an **admin**, I want to **ban users** so that I can enforce platform rules.
* As an **admin**, I want to **delete accounts** if they violate policies.

### Job Moderation

* As an **admin**, I want to **review job posts** so that fraudulent jobs are prevented.
* As an **admin**, I want to **remove inappropriate jobs** so that the platform remains safe.

### Company Moderation

* As an **admin**, I want to **approve company registrations** so that only real companies post jobs.

### Platform Monitoring

* As an **admin**, I want to **view statistics about users and jobs** so that I understand platform usage.

---

# 5️⃣ System Stories (Background Processes)

* As a **system**, I want to **send email verification after registration** so that accounts are confirmed.
* As a **system**, I want to **send application notifications** so that users stay informed.
* As a **system**, I want to **expire old job posts automatically** so that outdated jobs are removed.
